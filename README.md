# AdventOfCode
This repository contains solutions to the [Advent of Code](https://adventofcode.com) challenges.

## Description
Advent of Code is an annual event where participants solve a series of programming puzzles over the course of December. Each day presents a new challenge that requires problem-solving and coding skills. This repository includes my solutions to these challenges.
