# Advent of Code 2024
This folder contains solutions to the Advent of Code 2024challenges.

## Setup

1. This project is written in Rust. If you don't have Rust installed, you can get it by following the instructions on the [official website](https://www.rust-lang.org/tools/install).

2. Clone the repository:
    ```sh
    https://github.com/ayodejiige/AdventOfCode.git
    ```

## Usage
To run the solutions for each day's challenge using the appropriate command.

1. Go to the `aoc-2024` directory:
    ```sh
    cd path/to/your/aoc-2024
    ```
2. Run the solution for a specific day:
   ```sh
   # Syntax
   cargo run -- <day> path/to/your/aoc-2024/inputs

   # Example
   cargo run -- 1 path/to/your/aoc-2024/inputs
   ```
